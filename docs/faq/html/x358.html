How do I know if my patch got applied, and if not, why not? [GTK 2.x]

If you have used the GNOME bug tracking system (as stated above) then the status of your patch should be reflected in the bug report.

The GTK+ developers will generally add comments to the bug report stating what needs to be done to get the patch applied, or why the patch is not appropriate.
